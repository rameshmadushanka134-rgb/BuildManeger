# Shop Construction Cost Tracker (කඩ කාමර ඉදිකිරීම් වියදම් ලේඛනය)

A premium, easy-to-use, single-page web application designed to track and analyze construction costs, materials, labor, and payments for building two rental shops on your land.

---

## 🌟 Key Features

1. **Dashboard Overview (ප්‍රධාන දළ විශ්ලේෂණය):**
   - Live metrics: Total Expenses, Total Payments, Remaining Balances, and Budget Progress.
   - Dynamic charts showing category-wise expenses and shop comparisons.

2. **Shop Cost Division (කඩ කාමර අනුව බෙදීයාම):**
   - Categorize costs as **Shop 1 (Direct)**, **Shop 2 (Direct)**, or **Shared (පොදු)**.
   - The app automatically splits shared expenses 50/50 and sums up total allocations for each shop.

3. **Bilingual Support (සිංහල සහ English):**
   - Easily toggle the interface language between English and Sinhala (සිංහල) for ease of use.

4. **Data Privacy & Storage:**
   - Saved locally inside your browser's `localStorage`. No cloud backend or databases needed!
   - Back up your database anytime as a JSON file, or restore/move it to another browser/device.
   - Export expense tables directly into a Microsoft Excel-compatible CSV file.

5. **Printable Statements (PDF):**
   - The "Reports" tab is fully optimized for printing. Print to paper or save as clean, professional PDFs.

---

## 🚀 How to Run the Project

### Option A: Double-Click (Easiest)
Simply open your File Explorer, navigate to the project directory:
`e:\kapuwatta project\`
and double-click on [index.html](file:///e:/kapuwatta%20project/index.html) to open it in Google Chrome, Microsoft Edge, Firefox, or Safari.

### Option B: Local HTTP Server (Recommended for developers)
If you have Node.js installed, you can start a local development server from your terminal:
```bash
npx http-server .
```
Then open the displayed local link (usually `http://127.0.0.1:8080`) in your browser.
